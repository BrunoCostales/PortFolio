Sobre

Mí

Soy desarrollador web formado en una universidad privada de Argentina. Durante mis estudios obtuve una beca en una empresa reconocida del país, donde participé en el desarrollo de un sitio web como parte de un equipo. El proyecto se llevó a cabo de manera sincrónica, siguiendo metodologías ágiles como Scrum, con sprints, fechas límite y reuniones regulares que nos permitieron organizarnos y entregar con eficiencia.

Trabajé como Analista Técnico para una empresa de mensajería oficial en Argentina, donde brindé soporte a más de 4.000 usuarios, gestioné entornos Unix/Linux y desarrollé scripts utilizando PowerShell y CMD. Este rol también implicó una interacción constante con personas, fortaleciendo mis habilidades de comunicación y soporte al cliente.

Viví en Aspen, Colorado, donde desarrollé soluciones digitales personalizadas para negocios locales. Actualmente resido en Argentina, trabajando en proyectos personales y ampliando mis conocimientos. Trabajo con tecnologías como HTML, CSS, JavaScript, Java, Node.js, Express, MongoDB, MySQL, PHP, WordPress, Bootstrap, Git, GitHub y herramientas de scripting. Estoy comprometido con el aprendizaje continuo para ofrecer soluciones valiosas y de calidad.

Sobre Mí

Formación

Mi formación incluye una carrera en Desarrollo Web en una universidad privada de Argentina, donde adquirí habilidades tanto en desarrollo front-end como back-end, además de scripting. También realicé cursos especializados en desarrollo con WordPress y metodologías ágiles.

  • Desarrollador Freelance

    Mientras vivía en Aspen, Colorado, trabajé como desarrollador freelance y creé el sitio web LimitlessFeet.com para Foot Foundations, una empresa especializada en soluciones ortopédicas. Gestioné todo el ciclo de desarrollo, desde el diseño visual hasta la implementación técnica, fortaleciendo mi capacidad para trabajar de forma autónoma, comunicarme con clientes en inglés y entregar soluciones personalizadas alineadas con sus objetivos comerciales.

  • Analista de Software

    Trabajé como Analista de Software en Correo Argentino, el servicio postal nacional, brindando soporte técnico a una red de más de 4.000 usuarios en todo el país. Mis responsabilidades incluían la resolución de problemas en sistemas operativos, aplicaciones internas y conectividad. También desarrollé scripts de automatización utilizando PowerShell, CMD y entornos Unix/Linux para optimizar tareas administrativas y operativas.

  • Desarrollador Full Stack

    Como parte de un bootcamp intensivo en Digital House, desarrollé MyMad Games, una plataforma ficticia de e-commerce centrada en la venta de videojuegos. El proyecto fue construido utilizando HTML, CSS, JavaScript y Node.js, incorporando principios de diseño UX/UI y metodologías ágiles. Además de mejorar mis habilidades técnicas, este proyecto me enseñó a colaborar en equipo, cumplir con plazos ajustados y mantener una coherencia visual y funcional en una aplicación full stack.

    • Es una plataforma web moderna desarrollada para una empresa especializada en el escaneo 3D del pie y la creación de plantillas ortopédicas personalizadas.     MY MADGAMES

    VIDEO GAMES WEBSITE #2021

    • JavaScript
      HTML
      CSS
      NODEJS
      MYSQL

    • MyMad Games es una aplicación web de comercio electrónico totalmente funcional desarrollada durante el bootcamp intensivo de desarrollo full stack en Digital House. El sitio simula una tienda de videojuegos, permitiendo a los usuarios explorar productos, gestionar un carrito de compras y completar transacciones a través de una API de pagos integrada. El proyecto fue desarrollado en un entorno colaborativo utilizando metodologías ágiles como SCRUM, con el objetivo de replicar un flujo de trabajo real. La interfaz fue creada con HTML, CSS y JavaScript, utilizando Bootstrap para lograr un diseño responsivo. El backend se construyó con Node.js y Express, y los datos fueron almacenados y gestionados en una base de datos MySQL, asegurando una arquitectura sólida y escalable.
